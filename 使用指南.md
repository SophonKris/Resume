# 个人简历网站使用指南

## 🚀 快速开始

### 本地测试
1. 双击运行 `start-local-server.bat`
2. 浏览器访问 `http://localhost:8000`
3. 输入"田智宇"进行验证测试

### 服务器部署
1. 将整个项目文件夹上传到Windows Server
2. 在IIS管理器中创建网站，绑定域名 `ai.krisgeek.com`
3. 设置物理路径指向项目文件夹
4. 确保 `index.html` 为默认文档

## 📁 项目结构

```
个人简历网站/
├── index.html              # 验证页面（增强版安全）
├── resume.html             # 简历展示页面
├── css/                    # 样式文件
├── js/                     # JavaScript文件
├── web.config              # IIS配置文件
├── start-local-server.bat  # 本地测试服务器
└── 使用指南.md             # 本文档
```

- **浏览器指纹识别** - 防止清除数据绕过

## 🔧 故障排除

### 常见问题
- **404错误**: 检查IIS默认文档设置
- **验证失败**: 确保输入"田智宇"
- **页面不跳转**: 检查浏览器控制台错误
- **样式异常**: 确认CSS文件正确加载
